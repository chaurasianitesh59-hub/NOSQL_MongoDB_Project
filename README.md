## 📌 Overview

This project demonstrates MongoDB operations like CRUD, aggregation, joins (`$lookup`), filtering, and advanced queries using a student management dataset.

* faculty
* courses
* enrollments
* activities

## ⚙️ Setup

1. Open MongoDB Compass
2. Connect: `mongodb://localhost:27017`
3. Create DB: **Mongo Assignment**
4. Add collections & insert JSON data

* Complex filters & projections
* Aggregations & grouping
* Joins using `$lookup`
* Sorting & limiting
* Update, delete, upsert
* Array & nested queries
